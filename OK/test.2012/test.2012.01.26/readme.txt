Отдел Кадров от 23.01.2012 

Модуль "Кадры"

добавлены новые поля.

- В Основных данных и в Дополнительных данных по ИИАС по всем обязательным полям установлен значок «*»
  слева
- Ошибки, в которых указывается обязательные поля, которые не заполнены выводятся в текстовом виде (блокнот) 
   на экран, для возможности распечатать.
- Родственные отношения – Данные для ЦБ место жительства и место рождения если выбрали страну Узбекистан
  то не активные поля для ручного ввода, а если какую-либо другую страну,то не активные поля которые выбираются по справочнику ЦБ
- Сведения для работы – Основные данные – если стоит не «Шу банкда», то поля отдел (спр-к) и должность (спр-к) не активны
- При вводе нового сотрудника табельный номер генерируется автоматически, но есть возможность его корректировать и при этом проверяестся на уникальность
- При вводе нового сотрудника заполняются сразу две карточки "Основные данные" и "Данные для ИИАС"

для Халк банка:
- При Назначении, перемещении и увольнении сотрудника требуется подтверждение ГО, для этого в карточке сотрудника необходимо
  указать отдел,должность и нажать кнопку "Заявка", которая передаст данные в ГО для утверждения.(Только для Халк банка) 

- При отправки данных по сотруднику для утверждения ГО, установлена проверка на заполнения всех обязательных 
  полей по всем таб.номерам (Только для Халк банка)

- Таблица - сотрудники добавлена калонка "ИНН" 

Только для Халк банка:
- подтверждение нужно, если сотрудник новый,
  либо его текущая и/или будущая должность руководящие.
- кнопка активна, если получено подтверждение на новую должность,
  а сотрудник на нее еще не назначен или его текущая должность не руководящая.
- кнопка активна, если текущая должность не руководящая,
  но стоит проверка- назначение на руководящую должность без подтверждения невозможно.
- пока сотрудник не назначен на новую должность, 
  она(должность) отсутствует в сведениях о работе, иначе ее необходимо оттуда удалить.

Модуль "Кадры ГО" на единной базе

- Разработанно подтверждение в ГО с возможностью перехода в филиал, для просмотра данных по сотруднику (Только для Халк банка) 
- Разработан переход в любой филиал
